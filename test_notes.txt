Description: Provide commands to run the design 

python3 tools/run.py tests/dsl/quick_sanity.dsl.yaml

Function: 

1. Generates tests:
This command invokes toosl/dsl2tests.py to transalte user-provided DSL file (YAML) into per-IP JSON suites under 
tests/generated 

2.Builds the simulator (if not already built) 
Calls CMake to compile the SystemC framework and create build/sim/uvm_lite_sim

3. Runs the verification simulation 
Executes the compiled binary with SystemC, orchestrating the sequencer and agents
